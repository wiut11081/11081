# Restaurants list
In this web app you can find list of restaurants, and read feedback about them.

## How to open localy?
1. npm install - to instal all dependencies
2. npm start - to launch the project
3. go to the url: http://localhost:3000/

## Dependencies
- Express 
- SQLite
- EJS
- Bootstrap
